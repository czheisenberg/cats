# 产品需求文档 - 喵星人环游记

## 1. 产品概述
一个以全球地图为载体的猫咪品种展示网站，用户可以通过交互式世界地图探索世界各地的猫咪品种，点击地图区域或猫咪卡片进入详情页，了解每种猫咪的历史、特征和习性。

**目标用户**: 猫咪爱好者、宠物主人、对动物感兴趣的用户

## 2. 核心功能

### 2.1 页面模块
1. **首页（世界地图）**: 交互式全球地图，展示各地区代表性猫咪
2. **猫咪详情页**: 单一猫咪品种的详细信息展示
3. **地区列表页**: 按地区分类的猫咪列表

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | 世界地图 | SVG交互地图，支持缩放和平移，显示猫咪分布标记点 |
| 首页 | 地区弹窗 | 点击地图区域弹出该地区猫咪列表 |
| 首页 | 热门猫咪 | 展示热门/推荐的猫咪品种卡片 |
| 详情页 | 猫咪简介 | 名称、产地、体型、寿命等基本信息 |
| 详情页 | 外观特征 | 毛发类型、颜色、眼睛等详细描述 |
| 详情页 | 性格介绍 | 性情、习性、与人类互动特点 |
| 详情页 | 饲养建议 | 饮食、护理、运动需求 |
| 详情页 | 关联猫咪 | 同地区其他猫咪推荐 |
| 地区页 | 地区标题 | 地区名称和大洲图标 |
| 地区页 | 猫咪卡片列表 | 该地区所有猫咪品种网格展示 |

### 2.3 数据范围
**初始数据包含6个大洲15个国家/地区的30+种猫咪**:
- **亚洲**: 日本（江户短尾猫、Japanese Bobtail）、中国（狸花猫、狮子猫）、土耳其（安哥拉猫、Van猫）
- **欧洲**: 英国（英短、暹罗猫）、俄罗斯（俄罗斯蓝猫）、挪威（挪威森林猫）
- **非洲**: 埃及（埃及猫）、南非（非洲狮子猫）
- **北美洲**: 美国（缅因猫、美国短毛猫）、加拿大（斯芬克斯猫）
- **南美洲**: 巴西（巴西短毛猫）
- **大洋洲**: 澳大利亚（澳洲雾猫）

## 3. 用户界面设计

### 3.1 设计风格
- **主题**: 自然、温馨、探险感
- **主色调**: 
  - 主色: 温暖的琥珀色 #F59E0B
  - 辅助色: 森林绿 #059669 / 海洋蓝 #0284C7
  - 背景: 温暖的米白色 #FFFBEB，深色模式为深蓝灰 #1E293B
- **地图风格**: 手绘插画风格地图，避免过于写实的地图
- **卡片样式**: 圆角卡片，带有柔和阴影和悬停动效
- **猫咪卡片**: 包含猫咪照片、名称、产地标签
- **字体**: 
  - 标题: 圆润可爱的无衬线字体 (Nunito/Poppins)
  - 正文: 清晰易读的无衬线字体

### 3.2 动效设计
- 地图标记点: 呼吸动画，悬停时放大
- 页面切换: 平滑淡入淡出
- 卡片悬停: 轻微上浮 + 阴影增强
- 猫咪图片: 悬停时轻微缩放
- 加载状态: 骨架屏动画

### 3.3 响应式设计
- 桌面优先设计
- 平板: 地图缩小，保持功能完整
- 移动端: 地图可滚动，点击区域弹出底部抽屉

## 4. 核心交互流程

### 4.1 首页交互
```
用户进入首页 → 查看世界地图 → 悬停地图区域高亮 → 点击区域弹出猫咪列表 → 点击猫咪卡片进入详情页
```

### 4.2 猫咪详情页交互
```
进入详情页 → 查看猫咪基本信息 → 滚动查看详细信息 → 点击关联猫咪跳转 → 点击返回回到地图/列表
```

## 5. 技术需求
- 前端: Vue3 + TypeScript + Vite + Tailwind CSS + Vue Router
- 后端: NestJS + TypeScript
- 数据库: SQLite (开发) / PostgreSQL (生产)
- 地图: SVG交互地图组件
- 图片: 使用 Unsplash / Wikipedia 授权猫咪图片
